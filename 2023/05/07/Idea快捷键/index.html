Idea快捷键

在日常开发中,熟练的使用 IDE 的快捷键,可以大大提高我们开发效率,IDEA快捷键大全(持续更新)…

1.最常用的psvm,main生成main方法,sout生成输出

2.删除所在行:Ctrl+Y

3.新建,添加的快捷键:Alt + lnsert

在代码区使用Alt + lnsert会出现Generate菜单

在项目区使用Alt + lnsert会出现New

4.切换Java源文件:Alt + 右箭头或 Alt + 左箭头

5.目录项目关闭:Shift+ESC

6.运行:Ctrl + Shift + F10或者是Shift + F10,Debug:Shift + F9

7.提示方法的参数:Ctrl + p

8.切换窗口:Alt + (数字)

9.查找

快捷键 描述
Ctrl + F 当前文件中查找
Shift + Shift 快速查找
Ctrl + N 查找Class
Ctrl + Shift + N 查找File
Ctrl + Shift + Alt + N 查找Symbol
Ctrl + Shift + A 查找Actions

10.跳转切换

快捷键 描述
Ctrl + E 最近文件
Ctrl + Tab 切换文件
Ctrl + G 跳转指定行号

11.编辑和查看相关

快捷键 描述
Ctrl + Alt + T(经常用)
Shift + Enter 在当前行之后插入一行
Ctrl + Alt + Enter 在当前行之前插入一行
Ctrl + W 快速选中
Alt + 上下方向键 方法快速跳转
Ctrl + Z 撤销
Ctrl + Shift + Z 恢复
Ctrl + H 类的层级关系
Ctrl + D 快速复制(选中块也可以复制)
